Version 0.1.0 of the HPS AI Hub shipped with the minimalist editorial theme — a deliberate design system built on CSS custom properties, restrained typography, and zero JavaScript frameworks. The theme establishes the visual language for all subsequent content pages: monochrome palette, generous whitespace, and a mono-spaced eyebrow system for section numbering. PR #1 merged cleanly after passing the full SDD verify phase.
