Next.js is the primary frontend runtime. This skill enforces App Router conventions — server components by default, client components only when interactivity demands it. Server actions replace API routes for mutations. Folder structure follows the `(group)/page.tsx` pattern with co-located loading and error boundaries. Resend handles transactional email via a shared template library. Tailwind config is centralized with the HPS design token set.
