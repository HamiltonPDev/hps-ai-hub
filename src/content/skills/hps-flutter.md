Flutter handles native mobile and desktop apps across four HPS products, each with a distinct colour identity — teal, navy, green, orange. This skill enforces Riverpod as the sole state management solution, with providers organized by feature. Supabase Flutter client handles auth with deep link callbacks for magic links and OAuth. FCM push notifications follow a topic-based subscription model. The shared widget library uses atomic design principles with a strict import hierarchy.
